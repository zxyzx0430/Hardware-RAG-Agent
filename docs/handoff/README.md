# 交办单系统

每个线程开工时附带一个 TODO 清单，线程做完一项勾一项。

## TODO 格式

在每个 handoff 文档末尾加入：

`
## TODO 清单

- [ ] 任务一：xxx
- [ ] 任务二：xxx
- [ ] 任务三：xxx

---

### 规则
1. ✅ 完成一项 → 将 [ ] 改为 [x]
2. ➕ 线程执行中发现新问题 → 追加到 TODO 末尾，格式同
3. ❌ 判断为不需要做的 → 改为 [-] 并写明理由
4. 🔍 全部完成后 → 通知 00-control 审查，审查通过后删除该 handoff 文件
`

## 状态标记

| 标记 | 含义 |
|------|------|
| [ ] | 待做 |
| [x] | 已完成 |
| [-] | 跳过（写明理由） |
| [?] | 需确认（写明疑问） |
